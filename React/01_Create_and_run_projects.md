# 1. 프로젝트 생성

npm의 실행 도구인 npx를 사용해서 React 프로젝트를 생성할 수 있다네요!

설치를 원하는 경로에 들어가서

```shell

npx create-react-app 프로젝트이름

```

을 입력해주면 된다고 합니다..

`yarn`을 사용하고 싶다면?

```shell

yarn create react-app 프로젝트이름

```

이렇게 쓴다네요!
